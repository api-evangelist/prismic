# Prismic Plans

Prismic offers tiered subscription plans for individuals, teams, and enterprises.
Pricing is per repository and scales by number of users, locales, API call volume,
and bandwidth. All plans include unlimited documents and content types.

## Free

- Price: $0/month
- Users: 1
- Locales: 2
- API calls: 4 million/month
- CDN bandwidth: included
- Best for: Personal projects and proof-of-concept builds

## Starter

- Price: $10/month
- Users: additional users available
- Locales: more than Free tier
- API calls: 4 million/month
- Best for: Small sites and developer experimentation

## Small

- Price: $25/month
- Users: small team
- API calls: 4 million/month
- Best for: Small business websites

## Medium

- Price: $150/month
- API calls: 5 million/month
- Features: Advanced roles and permissions, multiple environments
- Best for: Growing marketing teams

## Platinum

- Price: $675/month
- API calls (soft limit): 10 million/month
- API calls (hard limit): 50 million/month
- Features: SLA, priority support, advanced environments
- Best for: High-traffic enterprise sites

## Enterprise

- Price: Custom (contact sales)
- Features: Custom API limits, dedicated support, custom SLA, advanced security
- Best for: Large organizations with complex multi-brand or multi-region needs

Source: https://prismic.io/pricing
